# CreateActivationKey

CreateActivationKeyはこのアプリケーションが実行されているPC固有の値(Cドライブのボリュームシリアルナンバー)から暗号化された文字列を生成。復号化する関数を実装する。
アプリケーションにロックをかけたい場合に使用する。


## Note
